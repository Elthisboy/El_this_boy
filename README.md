# 🌐 El_this_boy — Portfolio Website

Personal portfolio website for **Cristopher (El_this_boy)**, a Java Minecraft Mod Developer from Villa Alemana, Chile.

Built as a **single-page static site** with no frameworks, no bundlers, and no external dependencies — just clean HTML5 and CSS.

---

## 🔗 Live Preview

> Open `index.html` directly in any browser — no server required.

---

## 📁 Project Structure

```
elthisboyWeb/
├── index.html      # Full page markup
└── style.css       # All styles (organized by section)
```

---

## 📄 Sections

| Section | Description |
|---|---|
| **Hero** | Name, role, tagline, location, and CTA buttons |
| **Stats** | Key metrics: downloads, projects, collaborations, experience |
| **About** | Short bio and developer background |
| **Skills** | Grouped tags: languages, tools, and technical skills |
| **Projects** | Cards for mods, maps, and tools across GitHub and CurseForge |
| **Featured Collaborations** | YouTube thumbnail cards with "Watch on YouTube" links |
| **Experience** | Career summary and highlight metrics |
| **Media Kit** | What I offer, project types, and reasons to collaborate |
| **Contact** | Links to GitHub, CurseForge, and email |

## ⚙️ Tech Stack

- **HTML5** — semantic structure
- **CSS** — custom properties, grid, flexbox, transitions
- **No JavaScript** (except one inline line for the mobile nav toggle)
- **No frameworks** — zero dependencies
- **No build step** — open and done

## 🎨 Design System

| Token | Value |
|---|---|
| Background | `#0b0f1a` |
| Card | `#1a2235` |
| Accent | `#6366f1` |
| Accent hover | `#818cf8` |
| Text primary | `#e2e8f0` |
| Text muted | `#64748b` |
| Border radius | `12px` / `8px` |

**Responsive breakpoints:**
- `≤ 900px` — collaboration grid collapses to 1 column
- `≤ 768px` — mobile nav, 2-column stats, single-column projects
- `≤ 480px` — single-column buttons and contact cards
- `≤ 400px` — single-column stats
